# Issue 14: SARIF report generator

**优先级：P0** | **预估：M (human: ~2h / CC: ~15min)**

## 描述

实现 SARIF 2.1.0 报告生成器，输出符合规范的最小有效 JSON。

参考 ADR 0004：只填充 `runs[0].tool`、`runs[0].results`、`runs[0].invocations`，不做 taxonomies/rules 映射。

## 产出

- `src/scanner/report/sarif.ts`：`generateSarif(result: ScanResult): string`
- 输出 `sarifLog` 根对象，version `2.1.0`，`$schema` 指向 SARIF schema URL
- `runs[0].tool.driver.name` = `"miniapp-audit"`
- `runs[0].results[]` 从 `Finding[]` 映射：`severity` → `level`（critical→error, high→warning, medium→warning, low→note）、`filePath`/`line` → `locations[].physicalLocation`、`title`/`evidence`/`recommendation` → `message.text`
- `runs[0].invocations[0]`：执行信息（startTime、endTime、executionSuccessful）

## 验收
