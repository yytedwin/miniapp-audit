# Issue 15: HTML report generator

**优先级：P0** | **预估：M (human: ~2h / CC: ~20min)**

## 描述

实现自包含静态 HTML 报告生成器。生成的 HTML 文件内联所有 CSS/JS，双击浏览器即可打开，无需 Next.js 运行时。

参考 ADR 0004：HTML 报告的价值在于离线分享（发微信/邮件，对方直接打开）。

## 产出

- `src/scanner/report/html.ts`：`generateHtml(result: ScanResult): string`
- 完整的 HTML5 文档结构（`<!DOCTYPE html>` 到 `</html>`）
- 内联 CSS：暗色安全主题（与 Web UI 风格一致）、响应式布局、打印友好
- 报告内容：项目名、扫描时间、风险统计摘要（各级别数量）、按严重等级分组的 Findings 列表、每条 Finding 含文件路径、行号、证据、修复建议
- 空 Findings → 显示"未发现问题"页面
- 文件大小合理（100 条 Finding 不撑爆）
