# Issue 10: 实现 Web UI

**优先级：P1** | **预估：L (human: ~2d / CC: ~1h)**

## 描述

实现 Next.js Web UI，包含 4 个页面：
1. 扫描入口页 — 输入项目路径或拖拽上传 zip，选择项目类型，开始扫描按钮
2. 扫描中状态页 — 进度动画 + 适配器状态
3. 报告总览页 — 项目名、扫描时间、总体风险、按等级分组的 Findings 列表
4. 风险详情页 — 单条 Finding 的完整信息：证据、解释、修复建议、参考资料

## 产出

- `app/page.tsx` — 扫描入口
- `app/scan/page.tsx` — 扫描进度
- `app/report/[id]/page.tsx` — 报告总览 + 风险详情

## 依赖

- Issue 9 (CLI) — 共享同一扫描引擎

## 验收

- 首页能打开
- 能输入路径开始扫描
- 能看到扫描进度
- 报告页按风险等级分组展示
- 风险详情页包含证据和修复建议
- 移动端不炸
