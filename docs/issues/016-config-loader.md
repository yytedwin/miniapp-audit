# Issue 16: Config loader (miniapp-audit.config.json)

**优先级：P0** | **预估：S (human: ~1.5h / CC: ~10min)**

## 描述

实现配置文件加载器，支持从被扫描项目根目录读取 `miniapp-audit.config.json`。

参考 ADR 0004：配置文件只管"扫什么"——规则开关和排除路径。不管输出格式。

## 产出

- `src/scanner/config.ts`：
  - `loadConfig(projectPath: string): Promise<AuditConfig>`
  - `AuditConfig` 类型：`{ rules?: Record<string, { enabled: boolean }>, exclude?: string[] }`
- 无配置文件时返回默认值（所有规则启用，exclude 默认 `["node_modules", ".git", "dist"]`）
- 配置文件 JSON 解析失败 → 抛出明确错误
- `scanProject()` 使用 `loadConfig()` 结果过滤规则和排除路径
- 项目根目录新增 `miniapp-audit.config.json` 示例文件
