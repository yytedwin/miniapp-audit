# Issue 13: JSON report generator

**优先级：P0** | **预估：S (human: ~1h / CC: ~10min)**

## 描述

实现 JSON 报告生成器，将 `ScanResult` 直接序列化为 JSON。

参考 ADR 0004：JSON 报告不做中间 schema 转换，`ScanResult` 类型本身已是稳定的公开契约。

## 产出

- `src/scanner/report/json.ts`：`generateJson(result: ScanResult): string`
- 内部调用 `JSON.stringify(result, null, 2)`
- 输出包含完整的 `ScanResult`：项目路径、扫描时间、适配器结果、去重后 Findings
