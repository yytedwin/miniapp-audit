# Issue 18: Web scan history (localStorage)

**优先级：P0** | **预估：M (human: ~2h / CC: ~20min)**

## 描述

在 Web UI 中实现扫描历史记录功能，使用浏览器 localStorage 持久化。

参考 ADR 0004：历史列表保存摘要（项目路径、扫描时间、风险统计）；同时按 report id 保存完整 ScanResult 以支持跨会话回看报告。存储上限 50 条，淘汰时同步删除完整报告。数据不离开浏览器。

## 产出

- `app/history/page.tsx`：扫描历史列表页
  - 按时间倒序显示所有历史记录
  - 每条记录：项目路径、扫描时间、各级别 Finding 数量、总风险等级
  - 点击记录跳转到对应报告页
  - 支持删除单条记录
  - 空历史 → 显示引导文字
- 扫描完成后自动保存到 localStorage
- localStorage key 命名空间使用 `miniapp-audit:` 前缀
- 存储上限保护：超过 50 条时自动清理最旧记录
- 导航栏新增"历史记录"入口

## 依赖

None - 可立即开始（需要 Web UI 基础设施已就绪，当前已具备）

## 验收

- 执行扫描 → 刷新页面 → 历史记录中存在该条
- 点击记录 → 跳转到报告详情
- 删除记录 → 列表更新
- 超过 50 条 → 最旧记录被自动清理
- 空历史状态正常显示
