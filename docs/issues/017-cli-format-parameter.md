# Issue 17: CLI --format parameter

**优先级：P0** | **预估：S (human: ~1h / CC: ~10min)**

## 描述

扩展 CLI 入口，支持 `--format` 参数选择报告格式，支持 Markdown（默认）、JSON、HTML、SARIF 四种输出。

## 产出

- CLI 新增 `--format markdown|json|html|sarif` 参数
- `--output` / `-o` 参数根据 `--format` 自动调整扩展名（如 `--format html -o report.html`）
- `--format` 未指定时默认 `markdown`（向后兼容）
- 无效 format 值 → 输出错误信息并显示帮助
- 帮助信息更新，列出所有支持的格式

## 依赖

## 验收

- CLI 手动测试：`--format json`、`--format html`、`--format sarif` 均生成正确的对应格式文件
- `--format invalid` 显示错误
- 默认仍输出 Markdown
